# Solaris
Solaris _(codename until a better name is thought of)_ is an experimental idle game with crafting mechanics, developed using TypeScript with a strong focus on type safety and maintainable architecture. The goal is to create a relaxing yet engaging browser-based game where progress unfolds over time, and crafting plays a central role.


## 🎮 Gameplay Concept
- Crafting systems
- A visual crafting graph using React Flow
- Resource generation over time
- Passive and active gameplay loops

## 🛠 Tech Stack
- **TypeScript** – Deeply typed from core systems to UI
- **React Flow** – To visualize crafting chains and production flow
- **GSAP** – For smooth transitions and animation
- **Biome** - Linting and code quality
- **Vite** – Fast development and bundling

## 🎯 Project Goals
- Emphasize deep type safety with strict TypeScript
- Modular and maintainable codebase
- Explore scalable systems for idle gameplay mechanics
- Build a UI with accessible and minimal interactions
